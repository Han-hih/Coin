# BitPulse

> 관심있는 가상화폐를 검색하고 좋아요 목록 만드는 앱

## 작업환경
- 개발 기간: 24.04.07 ~ 24.04.15(8일, 진행중)
- 인원: 1인
- 최소 버전: iOS 17.0

## 주요기능
- 가상화폐 실시간 검색
- 실시간 등락률 및 현재가 제공
- 현재가와 거래량에 관한 정보 제공
- 좋아요 목록 생성 / 제거

## 기술 스택
- SwiftUI
- MVVM(Action - State)
- Combine, Charts
- URLSession, Async/Await, WebSocket
- Realm
